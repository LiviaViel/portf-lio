L-viaFlix: Plataforma de Filmes

L-viaFlix é uma plataforma web criada para facilitar a busca e a visualização de filmes. O projeto foi desenvolvido usando HTML, CSS e JavaScript, com foco em criar uma interface intuitiva e responsiva.

A página permite aos usuários navegar por diferentes gêneros de filmes, assistir trailers e acessar informações detalhadas sobre cada filme, como sinopse, elenco e avaliações.

O uso do JavaScript permite a interação dinâmica com o conteúdo, tornando a experiência do usuário mais fluida e agradável. Este projeto me ajudou a aprimorar minhas habilidades em desenvolvimento front-end e a entender melhor a criação de interfaces interativas.

Em Busca da Vila Mágica: Jogo Interativo

"Em Busca da Vila Mágica" é um jogo interativo que desenvolvi utilizando HTML, CSS e JavaScript. O objetivo é proporcionar uma experiência divertida, na qual o jogador explora uma vila encantada e resolve desafios para avançar na história.

O projeto combina elementos de design gráfico, programação e storytelling para criar um ambiente envolvente. Além disso, o jogo foi pensado para ser educativo, estimulando o raciocínio lógico e a criatividade.
